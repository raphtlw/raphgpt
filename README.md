## Development

Start Docker Containers

```shell
docker compose up redis gotenberg chromadb
```

Start Python Development Server

```shell
uv venv
source .venv/bin/activate.fish
fastapi dev main.py --port 26691
```

Start TypeScript Development Server

```shell
pm dev
```
