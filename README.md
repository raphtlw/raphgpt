# raphGPT 🤖

## About
Written in Typescript + Python.

## Features
- **Vision Capabilities**: I can read PDFs and accept ZIP files. Just send them my way and I'll process them for you.
- **Image Generation**: Using DALL-E 3 and Stable Diffusion, I can create cool images based on your prompts.
- **Songs & Speech**: Need a custom song or want me to read out text for you? I got you covered.
- **Math Whiz**: Got a tricky math problem? I'll break it down and solve it for you.
- **Google & Maps Search**: I can pull up relevant search results and help you find places on Google Maps.
- **Protective Spells**: I'm here to protect our conversations. No shady stuff allowed.

Runs in Telegram. No warranty guaranteed.
